## Usage

Start:

```
  DrawFrame **class_name**(**Mat image**);
```

Get Mat:

```
  **class_name**.GetMat();
```

Get start point:

```
  **class_name**.GetStart();
```

Get end point:

```
  **class_name**.GetEnd();
```
